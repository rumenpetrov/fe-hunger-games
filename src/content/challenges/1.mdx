## Requirements
* Create simple article UI for a blog page
* Content wrapper - 1200px
* Include headings from 1 to 6
* Include at least 8 paragraphs
* Use couple strong and em tags
* Use specific typography scle rule for the size of your text
* Include ordered and unordered list
* Include 1 quote
* Include a date for the post
* Include a photo and a figure
* Include at least 3 links
* Be mindful about the spacing around elements
* **[Special]** Create custom styles for the link by using the text decoration CSS properties.
